# Framework Utilities

## Round

Round Number To Decimal

```lua
Framework.Round(value, decimals)
```

- value: `number`
- decimals: `numbers`

returns: `number`

## FirstToUpper

Set First Character to uppercase

```lua
Framework.FirstToUpper(str)
```

- str: `string`

returns: `string`

## RandomString

Create A Random String

```lua
Framework.RandomString(num)
```

- num: `number`

returns: `string`

## RandomInteger

Create A Random String

```lua
Framework.RandomInteger(num)
```

- num: `number`

returns: `number`
